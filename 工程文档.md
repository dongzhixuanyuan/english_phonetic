# 工程文档 — 核心 SwiftUI View 功能说明

本文档汇总了 `EnglishPhoneticApp/Views/` 目录下核心 SwiftUI View 文件的功能与职责，便于快速了解各视图在 APP 中的定位。

---

## 1. ContentView.swift

**定位**：APP 的根视图入口。

**核心功能**：
- 初始化并持有唯一的 `TextbookViewModel`（`@StateObject`）。
- 使用 `NavigationStack` 包裹 `HomeView`，作为整个应用的导航起点。

**关键关系**：
```
ContentView
  └── NavigationStack
        └── HomeView (注入 viewModel)
```

---

## 2. HomeView.swift

**定位**：APP 首页，展示示例入口与“我的课本”列表。

**核心功能**：
- 顶部展示产品 Slogan 与标题。
- **示例区域**：若存在名为“示例课本”的数据，则展示示例单元与页面（横向滚动）。
- **我的课本区域**：列出用户创建的所有课本（排除示例课本），以卡片形式展示封面、页数、单元数、创建时间。
- **添加课本**：点击右上角 `+` 按钮，弹出 Alert 输入课本名称并创建。
- **导航跳转**：点击课本卡片后，通过 `navigationDestination` 进入 `TextbookDetailView`。

---

## 3. TextbookDetailView.swift

**定位**：课本详情页，管理单元与页面的增删改。

**核心功能**：
- 展示课本封面、名称、总页数、单元数。
- **添加单元**：点击按钮输入单元名称后创建。
- **单元折叠/展开**：每个单元可折叠，展开后显示该单元下的所有页面缩略图。
- **添加页面**：点击“添加页面”弹出 `ImagePickerSheet`，支持拍照或从相册选图；选图后自动进行 OCR 识别并生成音标标注。
- **页面管理**：每个页面缩略图右上角有 `Menu`，支持：
  - **重命名**：修改页面名称。
  - **删除**：删除页面并清理本地图片文件。
- **数据同步**：通过 `textbookId` 从 `ViewModel` 实时读取课本数据，确保添加/删除/重命名后页面即时刷新。

---

## 4. ReaderView.swift

**定位**：点读页，核心学习场景。

**核心功能**：
- **图片展示**：加载课本页面图片，支持双指缩放与拖拽浏览。
- **音标标签叠加**：在图片上根据 OCR 坐标渲染音标标签（`PhoneticLabel`）。
- **点击发音**：非编辑模式下，点击标签即可通过 `AVSpeechSynthesizer` 朗读对应单词。
- **编辑模式**：点击底部“编辑”进入校对模式，可：
  - **拖拽标签**：调整音标标签在图片上的位置。
  - **编辑标签**：点击标签弹出编辑面板，修改单词/音标；若音标为空，自动从本地词库匹配。
  - **添加标签**：在编辑面板中输入新单词，手动添加缺失的标注。
  - **删除标签**：移除错误的标注。
  - **重新识别**：对当前页面重新执行 OCR，重置所有标注。
- **底部工具栏**：提供编辑/完成、重识、重置缩放等操作入口。

**子视图**：
- `PhoneticLabel`：单个音标标签的 UI 渲染（含发音高亮、选中状态）。
- `AnnotationEditSheet`：标注编辑/新增的底部弹窗表单。

---

## 5. CameraPicker.swift

**定位**：系统相机与相册的桥接视图。

**核心功能**：
- `CameraPicker`：`UIViewControllerRepresentable` 封装 `UIImagePickerController`，`sourceType = .camera`，用于拍照。
- `PhotoLibraryPicker`：同样基于 `UIImagePickerController`，`sourceType = .photoLibrary`，用于从系统相册选图。
- 两者均通过 `@Binding var selectedImage: UIImage?` 将选中的图片回传给调用方。

---

## 视图流转关系

```
ContentView
  └── HomeView
        └── TextbookDetailView
              └── ReaderView

TextbookDetailView ──sheet──> ImagePickerSheet
                              ├── 拍照 ──> CameraPicker
                              └── 相册 ──> PhotoLibraryPicker

ReaderView ──sheet──> AnnotationEditSheet
```
